– Setup instructions

1. Install XAMPP / WAMP
2. Start Apache and MySQL
3. Open phpMyAdmin
4. Create a database named:
   onlinefoodorder
5. Import:
   online_food_order.sql
6. Copy project folder to:
   htdocs/ (XAMPP)
7. Open browser:
   http://localhost/project-folder/


– List of features implemented

Food Categories – Shows total number of categories available in the system.
Foods – Displays total number of food items.
Total Orders – Counts all orders placed in the system.
Revenue Generated – Calculates total revenue from all completed orders.
Pending Orders – Lists orders waiting to be processed.
On Delivery Orders – Tracks orders currently being delivered.
Cancelled Orders – Displays orders that were cancelled.
System Administrators – Shows total number of admin accounts.
Admin Navigation Menu – Quick access to key sections: Visit Site, Dashboard, Category Management, Food Items Management, Order Section, Manage Admin, Logout.
Dynamic Updates – Dashboard statistics and menu options update automatically as data changes in the system.
